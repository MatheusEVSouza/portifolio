Help Desk

Este projeto foi desenvolvido como parte de um curso na plataforma Udemy, com o objetivo de implementar um sistema de cadastro de chamados chamado 'Help Desk'. O sistema permite que os usuários cadastrem novos chamados e acompanhem o status dos chamados existentes, enquanto os administradores podem visualizar todos os chamados cadastrados e atribuí-los a membros da equipe. O sistema foi desenvolvido utilizando HTML, CSS e PHP, com o HTML sendo usado para criar a estrutura do sistema, o CSS para estilizar as diferentes seções do sistema e o PHP para implementar as funcionalidades de validação de login e interagir com um banco de dados para armazenar e recuperar os chamados cadastrados. As funcionalidades de validação de login incluem a validação de e-mails já cadastrados na aplicação para os usuários e uma validação especial para o login de administradores, que verifica se a senha inserida corresponde a um valor pré-definido. O sistema 'Help Desk' é uma solução simples e eficiente para o gerenciamento de chamados, fornecendo uma boa demonstração de como implementar um sistema de cadastro utilizando as tecnologias de desenvolvimento web mais populares.

DropBox Clone

O projeto desenvolvido durante o Curso Completo de JavaScript da HCode na Udemy tem como objetivo criar um clone da plataforma Dropbox, utilizando as principais tecnologias do mercado para desenvolvimento de aplicações web. A ideia é aprimorar o conhecimento em NodeJS, uma plataforma de desenvolvimento de aplicações em JavaScript, e também explorar as funcionalidades do Firebase, um banco de dados em tempo real da Google. Com o NodeJS, é possível criar um servidor web e desenvolver as principais funcionalidades para o armazenamento e gerenciamento de arquivos, enquanto o Firebase pode ser utilizado para armazenar e sincronizar os dados em tempo real entre dispositivos. A integração do Firebase no projeto permite que os arquivos sejam armazenados de forma segura e eficiente, garantindo que as informações estejam sempre disponíveis e sincronizadas para todos os usuários. Isso pode melhorar significativamente a experiência do usuário e tornar a plataforma mais eficiente. Além disso, o Curso Completo de JavaScript da HCode na Udemy é uma excelente opção para quem deseja aprender mais sobre programação web. O curso abrange desde os fundamentos do HTML, CSS e JavaScript até o desenvolvimento de aplicações web mais complexas usando NodeJS e Firebase, permitindo que os alunos adquiram habilidades valiosas para o mercado de trabalho. Com o curso, é possível se familiarizar com as principais tecnologias do mercado e desenvolver projetos desafiadores, como o clone da plataforma Dropbox.
